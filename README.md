# rugit

A git copy created with Ruby. Using this project to deep dive into how git works and teach myself Ruby in the process.

Env:

```shell
export PATH="$PWD/bin:$PATH"
```

Available commands:

- `rugit init`
- `rugit add`
- `rugit diff`
- `rugit commit`
- `rugit status` || `rugit status --porcelain`

Run tests: `rake`
